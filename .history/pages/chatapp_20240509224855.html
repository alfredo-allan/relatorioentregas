Register User




Esta página HTML contém um formulário simples para registrar um novo usuário. O usuário pode inserir um nome de usuário e marcar a caixa "Is Admin" se desejar registrar-se como administrador. Quando o formulário é enviado, ele faz uma solicitação POST para o endpoint '/register' no servidor Flask. Assim que o registro for concluído com sucesso, uma mensagem será exibida ao usuário. Certifique-se de substituir o endereço IP e a porta no código JavaScript pelo endereço correto do seu servidor Flask.
